## Introduction

> A demo of vuescroll, show the usage of `pull-to-refresh` and `push-to-load`.

## Features

- SSR
- Push to load
- Pull to refresh

## Usage

### Installation

```bash
   npm install
   # yarn install
```

### Run Server

```bash
  npm run dev
  # yarn serve
```

Plz visit `localhost: 8080`

### Genrate(SSR)

```bash
  npm run generate
  # Files will be generated in disk folder of root of project!
```
